This project is HTML and javascript based, and is optimized to look how intended in 1366 x 768 screen resolution (1360x768 works too) and on a maximized browser.


Everything in the folder is already where it should be. 
The breakdown of the folder is as followed: 
attack folder for attack animation gifs, 
audio folder for several audio cues and background music, 
blankpiece folder containing the artwork for the piece’s when they become darker and blurred out, 
images folder in regards to the character’s and some interface elements, 
and the projectfiles folder which contains game elements like the health bar, 
fully colored pieces and screen elements for the gallery.  


The index is where to start if you want to see everything the game has, 
the index is the title screen of the game where you make the decision black or red class.
